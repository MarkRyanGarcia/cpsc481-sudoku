# Sudoku Solver/Visualizer
## A Project for CPSC481 - Artificial Intelligence
Created by Mark Garcia and Elena Marquez

### Deployed @ [cpsc481-sudoku.vercel.app](https://cpsc481-sudoku.vercel.app/)
Source Code @ [github.com/MarkRyanGarcia/cpsc481-sudoku](https://github.com/MarkRyanGarcia/cpsc481-sudoku)

## Instructions:
1. Choose an algorithm in the dropdown menu.
2. Enter a puzzle OR click the "Generate Puzzle" button.
3. Hit "Solve".
4. Hit the "Reset" button, and repeat steps 1-3 until satisfied.
